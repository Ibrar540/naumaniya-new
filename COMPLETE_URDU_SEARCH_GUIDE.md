## ✅ ALL Search Operators Now Support Urdu!

Every search operator in the Admission Office now works in both Urdu and English with full comparison support.

## 🎯 Key Urdu Operators

| Urdu Operator | Meaning | Usage |
|---------------|---------|-------|
| `سے زیادہ` | Greater than / More than | Numbers, Dates |
| `سے کم` | Less than | Numbers, Dates |
| `سے بڑا` | Greater than / Bigger | Numbers, Dates |
| `سے چھوٹا` | Less than / Smaller | Numbers, Dates |
| `کے بعد` | After | Dates, IDs |
| `سے پہلے` | Before | Dates, IDs |
| `سے ... تک` | From ... to | Ranges |
| `درمیان ... اور` | Between ... and | Ranges |
| `سے اوپر` | Above | Numbers |
| `سے نیچے` | Below | Numbers |
| `سے آگے` | Ahead / Forward | IDs, Dates |
| `سے پیچھے` | Behind / Backward | IDs, Dates |
