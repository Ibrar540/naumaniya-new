# Budget Search Guide (Madrasa & Masjid)

## ✅ Comprehensive Search Implemented

Both Madrasa and Masjid budget screens (Income & Expenditure) now have intelligent search with full Urdu/English support.

## 🔍 Search Features

### 1. Amount Search (رقم)

| Operator | Urdu Examples | English Examples |
|----------|---------------|------------------|
| **Greater Than** | `رقم 1000 سے زیادہ`<br>`رقم 1000 سے بڑا`<br>`رقم 1000 سے اوپر` | `amount greater than 1000`<br>`amount > 1000`<br>`amount above 1000` |
| **Less Than** | `رقم 500 سے کم`<br>`رقم 500 سے چھوٹا`<br>`رقم 500 سے نیچے` | `amount less than 500`<br>`amount < 500`<br>`amount below 500` |
| **Between** | `رقم 500 سے 2000 تک`<br>`رقم درمیان 500 اور 2000` | `amount 500 to 2000`<br>`amount between 500 and 2000` |
| **Exact** | `رقم 1000` | `amount 1000` |

### 2. Date Search (تاریخ)

| Operator | Urdu Examples | English Examples |
|----------|---------------|------------------|
| **After** | `سال 2020 کے بعد`<br>`سال 2020 سے زیادہ` | `after 2020`<br>`date after 2020`<br>`since 2020` |
| **Before** | `سال 2020 سے پہلے`<br>`سال 2020 سے کم` | `before 2020`<br>`date before 2020`<br>`until 2020` |
| **Between** | `سال 2020 سے 2024 تک`<br>`درمیان 2020 اور 2024` | `from 2020 to 2024`<br>`between 2020 and 2024` |
| **Specific Year** | `سال 2023` | `year 2023`<br>`2023` |
| **Specific Month** | `جنوری`<br>`جنوری 2023` | `January`<br>`January 2023` |

### 3. Description Search (تفصیل)

Simply type any part of the description in Urdu or English:
- Urdu: `تنخواہ` / `کتابیں` / `بجلی`
- English: `salary` / `books` / `electricity`

## 📝 Complete Examples

### Amount Search Examples

#### Income (آمدنی)
```
رقم 5000 سے زیادہ          → Income greater than 5000
رقم 1000 سے 10000 تک      → Income between 1000 and 10000
رقم 2000 سے کم            → Income less than 2000
amount > 5000              → Income greater than 5000
amount between 1000 and 10000 → Income between 1000 and 10000
```

#### Expenditure (خرچ)
```
رقم 3000 سے زیادہ          → Expenditure greater than 3000
رقم 500 سے 5000 تک        → Expenditure between 500 and 5000
رقم 1000 سے کم            → Expenditure less than 1000
amount > 3000              → Expenditure greater than 3000
amount < 1000              → Expenditure less than 1000
```

### Date Search Examples
```
سال 2023                  → Entries from year 2023
سال 2020 کے بعد           → Entries after 2020
سال 2020 سے 2024 تک      → Entries from 2020 to 2024
سال 2020 سے پہلے          → Entries before 2020
جنوری 2023                → Entries from January 2023
جنوری                     → All January entries
after 2020                 → Entries after 2020
from 2020 to 2024          → Entries from 2020 to 2024
January 2023               → Entries from January 2023
```

### Description Search Examples
```
تنخواہ                     → All salary entries
کتابیں                    → All book entries
بجلی                      → All electricity entries
salary                     → All salary entries
books                      → All book entries
electricity                → All electricity entries
```

### Combined Search Examples
```
تنخواہ رقم 50000 سے زیادہ  → Salary entries > 50000
جنوری رقم 1000 سے کم      → January entries < 1000
سال 2023 رقم 5000         → 2023 entries with amount 5000
```

## 🎯 Key Urdu Operators

| Urdu Operator | Meaning | Usage |
|---------------|---------|-------|
| `رقم` | Amount | Amount searches |
| `سے زیادہ` | Greater than / More than | Numbers, Dates |
| `سے کم` | Less than | Numbers, Dates |
| `سے بڑا` | Greater than / Bigger | Numbers |
| `سے چھوٹا` | Less than / Smaller | Numbers |
| `کے بعد` | After | Dates |
| `سے پہلے` | Before | Dates |
| `سے ... تک` | From ... to | Ranges |
| `درمیان ... اور` | Between ... and | Ranges |
| `سے اوپر` | Above | Numbers |
| `سے نیچے` | Below | Numbers |
| `تاریخ` | Date | Date searches |
| `سال` | Year | Year searches |
| `مہینہ` | Month | Month searches |
